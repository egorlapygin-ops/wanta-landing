# Как запустить проект

## 1. Установите Node.js (если ещё нет)

- Скачайте LTS-версию с [nodejs.org](https://nodejs.org/) и установите.
- В терминале проверьте: `node -v` и `npm -v` — должны вывести версии.

## 2. Установите зависимости

В папке проекта выполните:

```bash
cd /Users/egorlapygin/Documents/Wanta
npm install
```

Дождитесь окончания установки.

## 3. Запустите сервер

```bash
npm run dev
```

В терминале должно появиться что-то вроде:

```
▲ Next.js 14.x.x
- Local:        http://localhost:3000
```

## 4. Откройте сайт в браузере

**Сайт сам не открывается** — откройте вручную:

- Перейдите по адресу: **http://localhost:3000**

Или нажмите ссылку `Local: http://localhost:3000` в терминале (если терминал это поддерживает).

---

## Если не открывается

- **«command not found: npm»** — Node.js не установлен или не добавлен в PATH. Переустановите Node.js с nodejs.org.
- **Ошибка при `npm install`** — проверьте интернет. Если ошибка про `node-gyp` или нативные модули — убедитесь, что установлены Xcode Command Line Tools: `xcode-select --install`.
- **Белый экран в браузере** — откройте DevTools (F12) → вкладка Console, посмотрите ошибки и пришлите текст.
- **Порт 3000 занят** — запустите на другом порту: `npm run dev -- -p 3001` и откройте http://localhost:3001.
